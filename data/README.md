# POLAR: Posture-level Action Recognition Dataset
   
   
## 1. Dataset Preparation
   
1. Download all the files. 
2. Unzip `annotations.zip` and `images.zip` in their corresponding directories.
3. The directory structure of the POLAR dataset is as follows:
   ```
   ./POLAR
   │  
   ├─Annotations
   │      p1_00001.json
   │      p1_00002.json
   │      p1_00003.json
   │      ...
   │      p1_35323.json
   │      p1_35324.json
   │  
   ├─ImageSets
   │      test.txt
   │      train.txt
   │      trainval.txt
   │      val.txt
   │      
   ├─JPEGImages
   │      p1_00001.jpg
   │      p1_00002.jpg
   │      p1_00003.jpg
   │      ...
   │      p1_35323.jpg
   │      p1_35324.jpg
   │      
   └─README.md
   ```
